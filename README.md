# GoPlan – Live Like A Local

GoPlan là website hỗ trợ du lịch Đà Nẵng, giúp người dùng tìm kiếm lịch trình, homestay, địa điểm trải nghiệm và tiện ích du lịch.

## Chức năng chính

- Khám phá địa điểm du lịch Đà Nẵng
- Xem lịch trình du lịch
- Gợi ý homestay
- Trải nghiệm địa phương
- Tìm tiện ích du lịch

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript

## Cấu trúc project

```bash
index.html
style.css
script.js
itinerary.html
homestay.html
experiences.html
utilities.html
```

## Cách chạy project

1. Clone repository:

```bash
git clone https://github.com/Hoshino22/GoPlan.git
```

2. Mở file `index.html` bằng trình duyệt.
